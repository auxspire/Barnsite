Baker's Barn - Structured Assets

Folder layout:
assets/brand/logo.jpg
assets/brand/poster.jpg
assets/hero/hero-1.mp4 (if provided)

assets/menu/<category>/<item>-1.jpg   (always present; placeholder generated if missing)
assets/menu/<category>/<item>-1.mp4   (present if a matching video existed)

assets/gallery/photos/g-###.jpg   (extra photos not mapped to menu items)
assets/gallery/videos/v-###.mp4   (extra videos not mapped to menu items)

Notes:
- All menu images were normalized to JPG for web compatibility.
- Any missing menu item image is generated as a light-grey placeholder with a greyed-out Baker's Barn logo.
- See MANIFEST.csv for the exact mapping from your original files to the output structure.
